# 🚀 Repvio

> Treine rápido. Acompanhe tudo.

Repvio é um SaaS mobile criado para personal trainers que querem montar treinos rapidamente e acompanhar o progresso dos seus alunos de forma simples e eficiente.

---

## 🧠 Sobre o projeto

O Repvio nasce com um foco claro:

**velocidade e simplicidade.**

Em vez de usar várias ferramentas diferentes, o personal pode centralizar tudo em um único lugar:

- Criar e gerenciar alunos
- Montar treinos em segundos
- Acompanhar evolução
- Registrar execuções de treino

---

## ⚡ Funcionalidades (MVP)

- 👤 Gestão de alunos
- 🏋️ Criação de treinos
- 📊 Acompanhamento de progresso
- ✅ Registro de execução de treinos

---

## 🛠️ Tecnologias

- Flutter
- Riverpod (gerenciamento de estado)
- Firebase (Auth + Firestore)

---

## 🎯 Objetivo

Ser o app mais rápido e intuitivo para personal trainers no dia a dia.
